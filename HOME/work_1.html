Вывести список пользователей имя и возраст

  • 1. Вывод осуществлять средствами React
  • 2. Для вывода создать 2 компонента - UserList, User
  • 3. Cписок users передать в компонент UserList через props
  • 4. В компоненте UserList используя итератор map вызывать компонент User которому через props передавать данные пользователя
  • 5. Вывод пользователей должен быть отсортирован по возрасту в поряке возрастания
